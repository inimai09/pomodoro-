# pomodoro-
This is a Pomodoro timer web app that helps you focus and stay productive.
It combines:

A timer for work/break sessions

Background chill music

Looping video background

Everything is interactive: start, pause, reset, next/previous songs, and the video syncs with the music.

# Technologies Used

HTML5

CSS3

JavaScript

HTML5 <video> & <audio> elements
